# Requirements

- **nodejs >= v14.7.0**
- **npm >= v6.14.7**
- **yarn >= v1.19.1**

Clone project and run test.

```
$ git clone https://github.com/TVTTranVanTu/lapstore-api
$ cd lapstore-api
$ yarn install or npm install
$ yarn start or npm start
```
